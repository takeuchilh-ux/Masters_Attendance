============================================================
  MasuTa! 打刻アプリ  Phase 3  - 事業所管理者向け
============================================================

【機能】
  - ダッシュボード : 今日の出欠状況（30秒自動更新）
  - シフト       : 月単位でシフトを作成・編集
  - 申請承認     : スタッフからの遅刻・有給申請を承認/却下
  - タッチログ   : フェリカタッチ記録の確認
  - スタッフ管理 : 事業所内スタッフの登録・編集

【起動方法（開発）】
  1. pip install -r requirements.txt
  2. python main.py

【exe ビルド方法】
  1. build.bat をダブルクリック
  2. dist\MasuTa.exe が生成される
  3. MasuTa.exe を各事業所 PC に配布

【ログイン情報】
  - Supabase Authentication のメール＋パスワードでログイン
  - アカウントは本部が Supabase ダッシュボードで作成
